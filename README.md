# AeroSim — Browser-Based 3D CFD Platform

Deploy to GitHub Pages by pushing all files to your repo root (main branch).
Then: Settings → Pages → Deploy from branch → main → / (root)

## Pages
- `index.html` — Dashboard
- `simulate.html` — 3D CFD Simulator
- `parts.html` — Parts Library
- `convert.html` — File Converter (DXF, IGES, STL → OBJ)
- `compare.html` — Side-by-side Comparison
- `physics.html` — Physics Reference

## All files are self-contained — no external CSS/JS files required.
## Only dependency: Three.js loaded from Cloudflare CDN in simulate.html.
